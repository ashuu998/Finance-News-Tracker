# Finance News Tracker 📈

Automated financial news sentiment 
analysis pipeline using Python & AI

## What it does
- Collects live financial news from NewsAPI
- Analyzes sentiment — Positive/Negative/Neutral
- Generates color-coded Excel report
- Shows market mood — Bullish/Bearish/Neutral

## Tech Stack
- Python
- Pandas
- NewsAPI
- TextBlob
- OpenPyXL

## How to run
1. Clone this repo
2. Install libraries: pip install -r requirements.txt
3. Add your API key in .env file
4. Run: python News_Tracker.py

## Output
- news_data.csv — Raw data
- news_report.xlsx — Color coded report
- Dashboard — Terminal output
